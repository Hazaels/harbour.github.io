More examples

More examples can be provided with the installation package of the Harbour. Look the /tests and /extras folders to find other samples, but the best examples are the source code of Harbour itself.

How to build/run sample applications:

  $ hbmk2 app.prg
  $ ./app
or run them as scripts: 
  $ hbrun app.prg
